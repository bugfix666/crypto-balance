## Домашнее задание: Система баланса пользователя

**Описание:**

Реализовать простую систему управления балансом пользователей. У каждого пользователя должен
быть баланс, который можно пополнять и с которого можно списывать средства. Каждое изменение
баланса должно фиксироваться в отдельной транзакции. Также должна быть возможность получить
список транзакций для конкретного пользователя.

**Функциональные требования:**

1. **Пользователи:**
    o Стандартные пользователи (без авторизации/регистрации — можно эмулировать).
    o У каждого пользователя должен быть уникальный идентификатор (например, UUID или
       ID).
2. **Баланс:**
    o Возможность пополнить баланс пользователя на заданную сумму.
    o Возможность списать с баланса заданную сумму (при условии, что средств достаточно).
    o Баланс должен храниться в базе данных например PostgreSQL.
3. **Транзакции:**
    o Каждое изменение баланса должно записываться в таблицу транзакций с указанием:
       § ID пользователя
       § Тип операции (credit или debit)
       § Сумма
       § Дата и время операции
    o Возможность получить список всех транзакций конкретного пользователя (от новых к
       старым).

**Технические требования:**

- PHP >= 8.1+
- Использование фреймворков не обязательно(предпочтительно Laravel).
- БД: например PostgreSQL.
- Взаимодействие через CLI или простейший HTTP-интерфейс (напр. Slim, Laravel Routes, или
    plain PHP).
- Код должен быть разделён на логические части (модели, сервисы, контроллеры при
    необходимости).

**Дополнительно (по желанию):**

- Покрыть базовую логику юнит-тестами.
- Использовать DI-контейнер (если применимо).
- Обработку ошибок и валидацию данных.

**Срок выполнения:** 1 – 3 часа.

**Формат сдачи:**

- Репозиторий на GitHub/GitLab или архив с проектом.
- Инструкции по запуску (README.md), включая настройку базы данных.


## Запуск

Чтобы запустить задание, необходимо запустить приложение Docker или запустить службу docker

затем выполнить команду

```
make up
```

## Команды

### Внести деньги на счет
```
wallet:deposit <amount> <wallet>
```
amount - сумма<br/>
wallet - UUID счета


### Вывести деньги со счета
```
wallet:withdraw <amount> <wallet>
```
amount - сумма<br/>
wallet - UUID счета


### Список пользователей с их кошельками
```
wallet:list
```

### Список транзакций пользователя
```
wallet:operations <user>
```
user - UUID пользователя

## Решение проблем
Если появится ошибка ниже, необходимо указать корневую папку проекта в приложении Docker в Resources -> Sharing и перезапустить сборку снова, используя ту же команду, что указана выше

Ошибочный ответ от демона: Mounts denied:
Путь /www/balance не является общим с хоста и не известен Docker.
Настроить общие пути можно в Docker -> Preferences... -> Resources -> File Sharing.
